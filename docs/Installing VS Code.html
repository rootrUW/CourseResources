Module 01: Visual Studio Code (VS Code) Installation Guide

This page provides step-by-step instructions for installing Visual Studio Code (VS Code) on Windows and macOS, plus troubleshooting tips and quick micro-labs to verify that everything works.

11.0 Installing Visual Studio Code (VS Code)

VS Code is the editor used in this course for writing SQL, Python, and documentation. It runs on Windows and macOS and is free to install.

Use the sections below that match your operating system. At the end, review the troubleshooting section if you run into any issues.

11.1 Install VS Code on Windows

Step 1: Download VS Code

  1. Open your browser and go to https://code.visualstudio.com/.
  2. Select Download for Windows.

Step 2: Run the Installer

  1. Locate the downloaded file, usually named VSCodeUserSetup-x64.exe.
  2. Double-select the installer.
  3. If prompted by Windows, select Yes.
  4. Accept the agreement and select Next.

Step 3: Choose Installation Options

  1. Keep the default installation location.
  2. Select the following checkboxes:
    • Add to PATH
    • Register Code as an editor for supported file types
  3. Select Next, then Install.

Step 4: Launch VS Code

  1. Select Finish to launch VS Code.
  2. You should see the Welcome screen.

Step 5: Install Required Extensions

  1. Select the Extensions icon.
  2. Search for ms-sql and install the SQL Server extension.
  3. (Optional) Install the Python extension.

Micro-Lab 11.1 (3 minutes): Verify Windows Installation

  1. Select Terminal > New Terminal.
  2. Type: 
    code --version
  3. If version info appears, VS Code is installed correctly.

11.2 Install VS Code on macOS

Step 1: Download VS Code

  1. Go to https://code.visualstudio.com/.
  2. Select Download for macOS.

Step 2: Move VS Code to Applications

  1. Open the downloaded ZIP file.
  2. Drag Visual Studio Code.app into your Applications folder.

Step 3: Open VS Code

  1. Open Launchpad or the Applications folder.
  2. Select Visual Studio Code.
  3. If macOS shows a security warning, select Open.

Step 4: Install Required Extensions

  1. Select the Extensions icon.
  2. Search for SQL Server (mssql).
  3. Install the extension.

Step 5 (Optional): Add VS Code to Terminal PATH

  1. Select View > Command Palette.
  2. Type Shell Command: Install 'code' command in PATH.
  3. Select it.
  4. Restart Terminal.

Micro-Lab 11.2 (3 minutes): Verify macOS Installation

  1. Open Terminal.
  2. Type: 
    code --version
  3. If version info appears, installation is complete.

11.3 Tips for Using VS Code in This Course

  • Use File > Open Folder to open assignment folders.
  • Use the Explorer panel to navigate files.
  • Use Terminal > New Terminal to run commands.
  • Use the Extensions panel to install tools.
  • Select View > Output for debugging information.

11.4 Troubleshooting VS Code Installation

Windows: "code is not recognized"

This means VS Code is not in the PATH.

  1. Select View > Command Palette.
  2. Run Shell Command: Install 'code' command in PATH.
  3. Restart PowerShell or Command Prompt.

macOS: "Cannot be opened because the developer cannot be verified"

This is macOS Gatekeeper blocking the app.

  1. Open System Settings.
  2. Choose Privacy & Security.
  3. Select Open Anyway next to VS Code.

Extensions will not install

Try these steps:

  • Check your internet connection.
  • Restart VS Code.
  • Make sure your system clock is correct.
  • If on a corporate network, confirm proxy settings.
